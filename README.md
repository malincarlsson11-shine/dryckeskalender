---

##  Privacy & anonymity

Privacy is a fundamental principle of this project.

- All data is stored **only locally in the user's browser**
- No data is sent to any server
- No login, no accounts, no personal data
- No third-party analysis or tracking
- The operator has no access to user data

### Same device
The calendar is linked to the device and browser where it was created.  
If browser data is cleared or the device is replaced, the calendar cannot be restored.

### Automatic deletion
For increased privacy, the calendar is deleted if no new entries have been made in 5 days.  
The deletion occurs when the user opens the calendar again.

##  Purpose and use

The purpose of this registration is to:
- give the user a clear overview of their alcohol consumption
- support self-reflection
- serve as a basis for discussions, for example with a therapist

The tool is not intended as a diagnostic or medical decision support tool.
